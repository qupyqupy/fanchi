1. 到http://ffmpeg.zeranoe.com/builds/ 下載ffmpeg-static壓縮檔。
  2. 因為ffmpeg的介面是屬於CLI，為了方便日後使用，要設定windows環境變數。
    先解壓縮檔案，得到的內容物如下圖：
  3. 新增資料夾C:\Program Files\ffmpeg\將內容物複製到裡面，方便日後管理。
  4. 在我的電腦按右鍵，點選內容
  5. 點選進階系統設定開啟系統內容面板，點選環境變數。
  6. 在系統變數區塊點選新增開啟新增系統變數面板。
    變數名稱：FFMPEG_HOME
    變數値：C:\Program Files\ffmpeg\bin
  7. 在系統變數區塊內找到Path變數，選取後點選編輯開啟面板。
    在變數輸入框最後加上;%FFMPEG_HOME%
  8. 設定完環境變數要重啟電腦讓它生效。
  9. 製作bat檔案方便之後轉檔，新增一個純文字檔，把下列指令貼入後存檔為.bat檔案。 
    set input=%1
set output=build/%~n1%~x1
echo =========== start ==============
mkdir build
ffmpeg -i %input% %output%
echo =========== over ==============
pause
  10. 直接把要轉檔的影片拖曳到.bat檔上就可以進行轉檔了，轉好的檔案會在原影片資料夾的build資料夾中。
